
# Stock Price Prediction

This project uses Linear Regression to predict stock prices based on historical data. It is a simple machine learning model that helps understand the basics of regression.

## Tools Used
- Python
- Jupyter Notebook
- Scikit-learn
- Pandas
- Matplotlib
